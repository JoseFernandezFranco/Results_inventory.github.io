# Portal de resultados EYSA

Este directorio reúne la portada principal, los índices por estudio y los materiales técnicos utilizados para construir los análisis.

## Navegación

- `index.html`: portada del portal.
- `assets/`: hoja de estilos y logo local de EYSA.
- `studies/`: páginas índice por estudio y copia de sus resultados originales.
- `materiales/`: CSV de entrada, scripts utilizados y ZIP directo para reproducir los estudios.
- `inventario-trip-proxy-lisa/index.html`: índice navegable de Proxy por trayectos.
- `inventario-traffic-proxy-lisa/index.html`: índice navegable de Proxy por tráfico.
- `estudio-temporal-inventario-contaminacion/index.html`: índice navegable de Estudio temporal.

## Reejecución técnica

Consulta `materiales/README_ejecucion.md` o descarga `materiales/reproducir_estudios_8_archivos.zip`.
